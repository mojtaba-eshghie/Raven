## Analysis / Plotting of results

This folder contains scripts and resources related to the plotting of invariants from our analysis.



## Content Overview

| Name                | Description                                                                 |
|--------------------------|-----------------------------------------------------------------------------|
| `plotting.ipynb`       | Plots all plots in my thesis. |
| `visualization` | Results of the plottings for this thesis.  |



## Usage
To generate or update the plots:

- Make sure your data is prepared and accessible to the script.

- Run the plotting.ipynb script. This will output the updated plots into the visualization/ directory.
